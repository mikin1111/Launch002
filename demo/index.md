# 设计规范总览 · Design Guidelines Demo

> 15 个组件的规范预览，通过顶部 Tab 切换查看各组件详细页面。

## 包含组件

| 序号 | 组件名称 | 文件 | 说明 |
|------|---------|------|------|
| 1 | Color | `color.html` | 颜色 Token 预览（Light / Dark） |
| 2 | Font | `font.html` | 字阶规范预览 |
| 3 | Spacing | `spacing.html` | 间距规范预览 |
| 4 | Icon | `icon.html` | 图标预览（全部） |
| 5 | Divider | `divider.html` | 分割线规范预览 |
| 6 | Button | `button.html` | 按钮组件规范演示 |
| 7 | Checkbox | `checkbox.html` | 复选框组件规范预览 |
| 8 | Tabs | `tabs.html` | Tabs 组件规范预览 |
| 9 | Tag | `tag.html` | Tag 组件规范预览 |
| 10 | Select | `select.html` | Select 组件规范预览 |
| 11 | Search | `search.html` | Search 组件规范演示 |
| 12 | FAQ | `faq.html` | FAQ Accordion 组件 |
| 13 | Header/Footer | `header-footer.html` | Header / Footer 规范预览 |
| 14 | CoinTitle | `cointitle.html` | 币种标题组件 |
| 15 | NumberView | `numberview.html` | 数值显示组件 |

## 使用方式

在浏览器中打开 `index.html`，通过顶部 Tab 导航栏切换查看各组件的详细规范预览页面。

- 支持键盘左右方向键切换 Tab
- 每个子页面保留各自独立的样式和交互
- Tab 栏支持横向滚动（组件较多时）
